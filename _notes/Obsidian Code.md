# Obsidian Code

> [!info]
> Claude AI를 Obsidian에 직접 통합하는 플러그인

## 프로젝트 개요

**Obsidian Code**는 Claude AI를 노트 작성 애플리케이션에 직접 통합하는 Obsidian 플러그인입니다. 단순한 챗봇이 아니라, Claude가 사이드바에서 작동하면서 Vault 콘텐츠를 적극적으로 관리하고 수정할 수 있습니다.

## 주요 기능

### 🎯 스마트 노트 핀닝
사용자가 노트를 고정하면 여러 파일을 열어도 해당 노트가 접근 가능하게 유지되어 컨텍스트를 보존합니다.

### 🔐 권한 모드
세 가지 작동 모드 존재:
- **AUTO**: 자동 실행
- **SAFE**: 작업 승인 필요
- **PLAN**: 실행 없이 계획 수립

### 🧠 Obsidian 스킬 통합
Claude가 다음을 이해하도록 원클릭 스킬 설치로 활성화:
- Wiki-links (`[[note]]`)
- Embeds (`![[file]]`)
- Callouts (노트, 경고 등)
- YAML Frontmatter

### 📎 다중 파일 컨텍스트
"@" 심볼로 여러 파일을 동시에 첨부 가능하며, 칩 형태로 빠르게 관리할 수 있습니다.

### ✏️ 직접 수정
Claude가 적절한 보안장치를 통해:
- 노트 생성, 편집
- 터미널 명령 실행
- Vault 내용 관리

## 기술 요구사항

플러그인이 정상 작동하려면:
- Anthropic의 **Claude Code CLI** 사전 설치 필요
- 브라우저 기반 로그인으로 인증

## 설치 방법

### 1. BRAT (권장)
자동 업데이트 및 설치가 간편합니다.

### 2. 수동 설치
저장소를 직접 복제하고 빌드하는 개발자 설치 방식입니다.
